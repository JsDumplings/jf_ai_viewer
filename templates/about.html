关于九峰DICOM阅片工作站

This app is based on the Cornerstone open source project. All 2D image rendering is done on the client side side using HTML5 and JavaScript. DICOM P10 Instances are retrieved via WADO, parsed and displayed by the viewer - the server side performs no processing.

Product: JF Dicom Viewer
